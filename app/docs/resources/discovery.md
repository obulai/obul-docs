# Discovery

Discover x402-enabled APIs and services that work with Obul.

Obul provides a curated discovery layer for the x402 economy — find, evaluate, and integrate pay-per-use APIs without managing wallets or crypto.

## What You'll Find

- **Web Scraping** — Firecrawl, Browserbase, Zyte, Minifetch, and more
- **Web Search** — Exa, Perplexity, Tavily, Jina, and others  
- **Lead Enrichment** — Apollo, Hunter, StableEnrich, Coresignal
- **Social Media** — X/Twitter, Neynar, Reddit integrations
- **Blockchain/DeFi** — CoinGecko, Zapper, HeyElsa, Ordiscan
- **Media Generation** — Image, audio, and video generation APIs
- **Security & Risk** — Smart contract auditing and risk analysis

## How Discovery Works

1. **Browse** the [API Marketplace](/resources/marketplace) to find services
2. **Evaluate** pricing and capabilities before integrating
3. **Connect** with a single API key — no separate accounts needed
4. **Pay** per-use through consolidated Obul billing

## Getting Started

No setup required. Every API in the marketplace works immediately with your `OBUL_API_KEY`.

```bash
curl -H "X-Obul-Api-Key: $OBUL_API_KEY" \
  "https://proxy.obul.ai/proxy/https/firecrawl.x402endpoints.com/v1/scrape" \
  -d '{"url": "https://example.com"}'
```
